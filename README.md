# Otto 🤖

## O que é?

Otto é um assistente inteligente desenvolvido para conversar, aprender com o usuário, organizar projetos, executar tarefas e atuar como um companheiro digital.

## Objetivos

- Conversar naturalmente.
- Aprender com o usuário (com autorização).
- Organizar projetos.
- Executar tarefas.
- Ler documentos.
- Criar planilhas.
- Programar.
- Pesquisar na internet.
- Evoluir com novos módulos.

## Tecnologias

- Python
- FastAPI
- React
- SQLite (inicialmente)
- PostgreSQL (futuro)
- OpenAI API
- Ollama (modelos locais)
